Project 2

EYYYY, IM WALKIN HERE!

Project 2 involved adding inverse kinematic functionality to our robot from project 1. This project was built off of the source code from project 1 and continued to be developed using javascript. This project also updated the source to use transformation matrices instead of functions for the position of the arms.

Running Instructions: Open page.html in the browser and the project will be loaded, no compilation is required to run this project.
