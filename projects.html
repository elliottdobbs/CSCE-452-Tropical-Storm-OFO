Project 5

Murry Chrimus

The goal of project 5 is to implement the cell decomposition method of robotic motion planning. There is a green point representing the robot start point, and a red point representing the end point. The boxes and the points can be clicked and moved around. We used the same base for our program, utilizing the libraries and functions written for previous projects. The program was written in javascript.

Running Instructions: Open page.html in the browser and the project will be loaded, no compilation is required to run this project.
